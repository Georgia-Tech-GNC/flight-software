# HALAL

This directory contains the Hardware Abstraction Layer Abstraction Layer, or HALAL for short. The HALAL contains a collection of abstract hardware types, from sensors such as barometers and IMUs to storage devices such as SD cards and flash chips. Each hardware type is also intended to have a 'Fake', or software-emulated version in some cases powered by fake data.

I have created a few files as a template to show how the directory structure should be. Feel free to delete these and create your own.
