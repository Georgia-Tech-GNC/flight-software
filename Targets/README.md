# Targets

This directory contains the specific hardware configuration needed for different (dev) boards
STM32CubeBullshit is a standin for the stuff generated by STM32's unholy but ever helpful CubeMX software.
In practice, the folder generated by STM32CubeMX will be the root directory for each target. 
Don't forget to include port_config.h and config.h though. See the actual files in the example folders for info on what they are
